# Password Security Audit Suite

## Project Overview
This project is a Python-based Password Security Audit Suite that helps evaluate password security.

## Features
- Password Strength Analysis
- Dictionary Generator
- Hash Generator (MD5, SHA-256, SHA-512)
- Brute Force Time Estimation
- PDF Audit Report Generation

## Requirements
Python 3.x

Install dependencies:

```bash
pip install reportlab
```

## Run

```bash
python main.py
```
